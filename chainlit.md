# Bodden-Bot

![Bodden-Bot Icon](/public/bodden-icon.png)

Willkommen beim Bodden-Bot.

Ihr KI-Assistent fuer strafrechtliche Analysen.

## So funktioniert es

1. **PDF hochladen**: Laden Sie eine Anklageschrift, ein Urteil oder ein anderes strafrechtliches Dokument hoch.
2. **Automatische Analyse**: Der Agent extrahiert den Text, identifiziert relevante Straftatbestaende und zerlegt die Pruefung in logische Schritte.
3. **Gutachten erhalten**: Sie erhalten ein strukturiertes Gutachten im Gutachtenstil mit Quellen aus Kommentarliteratur und aktueller Rechtsprechung.

## Hinweis

Dieses Tool dient ausschliesslich zu Informationszwecken und ersetzt keine anwaltliche Beratung.
